# WNCDamHazardPotential

## This Web map is a Bivariate map focusing on the Hazard potential of dams in western North Carolina after Hurricane Helene hit the region in 2024. It uses Javascript, CSS, and HTML and includes interactive popups and a zoom in and our option to let the user find specific dams in the region. Data is sourced from the NOAA and NC DEQ. Geojson files can be found in the assets folder.
